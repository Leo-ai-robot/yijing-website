# 易学网站 · 轻量化工具产品方案

---

## 一、整体架构

### 1.1 工具入口

易学工具栏目下拉菜单结构：

```
易学工具 ▾
  ├─ 每日宜忌     → daily-fortune.html
  ├─ 干支转换器   → ganzhi-converter.html
  ├─ 简易起卦     → divination.html
  ├─ 五行格局     → wuxing-chart.html
  └─ 节气对照     → solar-terms.html
```

点击"易学工具"主导航本身 → 跳转到工具总览页 tools.html（展示5个工具卡片，每个卡片含名称、一句话描述、入口按钮）。

### 1.2 所有工具共同原则

- **纯前端实现**：零后端，所有计算逻辑在 JS 中完成，数据硬编码或 JSON 嵌入
- **共享数据层**：从现有 js/main.js 提取 `tiangan`、`dizhi`、`wuxingMap`、`shengxiao` 等数据，构建独立 `tools-data.js` 工具数据模块，所有工具共用
- **单页独立运行**：每个工具一个独立 HTML 页面，不依赖框架
- **复用现有CSS变量**：`--cinnabar`、`--ink-black`、`--paper`、`--border-light` 等
- **页面底部的"上一节/下一节"**：替换为工具间的切换链接
- **文化声明**：每个工具底部加一行声明"仅供传统文化学术研究参考"

### 1.3 页面结构模板（每个工具页通用）

```
[标准网站Header + 导航]
  ↓
[页面Banner] → 工具名称 + 面包屑
  ↓
[工具主区域] → 交互控件 + 结果展示区
  ↓
[文化声明] → 一行小字
  ↓
[工具切换导航] → 其他4个工具的快速跳转
  ↓
[标准Footer]
```

### 2.1 每日宜忌 (daily-fortune.html)

**核心功能**
根据当日干支日期，自动生成传统黄历宜忌信息。

**用户场景**
用户每天打开网站，快速查看今天适合做什么、不宜做什么。入门级用户的日常互动入口。

**数据来源**
- 从首页干支日历的 JS 逻辑中获取今日干支日期
- 内置宜忌规则库（基于干支/五行/建除十二值等，但简化到纯前端可维护的级别）

**交互逻辑**

| 步骤 | 交互 | 反馈 |
|------|------|------|
| 页面加载 | 自动获取当天日期 | 显示公历日期 + 干支日 + 生肖 |
| 主动刷新 | 点击"刷新"按钮 | 实际就是重新加载当天数据 |
| 切换日期 | 点击"前一天/后一天"按钮 | 日期变化、干支和宜忌同步更新 |

**页面布局**

```
┌─────────────────────────────────────┐
│  今日宜忌 · 2026年5月11日            │
│  辛卯日 · 金木 · 生肖马              │
│                                     │
│  ┌──────────┐  ┌──────────┐        │
│  │ 宜        │  │ 忌        │        │
│  │          │  │          │        │
│  │ ✓ 祭祀   │  │ ✗ 嫁娶   │        │
│  │ ✓ 出行   │  │ ✗ 开市   │        │
│  │ ✓ 会友   │  │ ✗ 动土   │        │
│  │ ✓ 求医   │  │          │        │
│  └──────────┘  └──────────┘        │
│                                     │
│  冲煞：冲蛇 · 煞西                   │
│  吉神：天德 · 月德 · 驿马            │
│                                     │
│  ◀ 前一天  │  刷新  │  后一天 ▶     │
├─────────────────────────────────────┤
│  仅供传统文化学术研究参考            │
```

**前端实现要点**
- 数据量：约60组干支 × 每条干支对应5-8宜/忌项，可硬编码
- 也可用简单规则引擎：如"木日宜祭祀、火日宜求医"等通用规则 + 特殊日例外
- 推荐数据量控制在 60 × 15 = 900 条以内，直接 JSON 嵌入

### 2.2 干支纪年转换器 (ganzi-converter.html)

**核心功能**
公历日期 ↔ 干支日期的双向实时转换。

**用户场景**
用户想知道某个生日、纪念日或其他特定日期对应的干支是什么；或者看到"甲子年"想知道对应哪一年。

**交互逻辑**

| 操作 | 交互 | 反馈 |
|------|------|------|
| 点选公历日期 | 日期选择器 onChange | 立即显示年/月/日/时四柱干支、生肖、纳音五行 |
| 输入年份查询干支 | 输入公历年份 | 显示该年干支、生肖、五行属性 |
| 选择干支查公历 | 下拉选择天干+地支 | 显示该干支对应的年份范围（如甲子 = 1984, 2044...） |

**页面布局**

```
┌─────────────────────────────────────┐
│  干支纪年转换器                      │
│                                     │
│  模式切换： [日期→干支] [干支→年份]  │
│                                     │
│  ──── 公历 → 干支 ────              │
│  📅 选择日期: [2026-05-11 ▼]        │
│                                     │
│  年柱: 丙午 (马)  · 天河水           │
│  月柱: 乙巳 (蛇)  · 覆灯火           │
│  日柱: 辛卯 (兔)  · 松柏木           │
│  时柱: （需选时辰）                   │
│                                     │
│  五行局: 火 土 木                    │
│                                     │
│  ──── 干支 → 年份 ────              │
│  天干: [丙▼]  地支: [午▼]           │
│  对应年份: 2026, 1966, 1906...      │
├─────────────────────────────────────┤
│  仅供传统文化学术研究参考            │
└─────────────────────────────────────┘
```

**前端实现要点**
- 日干支计算：已有公式（首页 JS 用的基准日 2000-01-01 = 甲子日），可直接复用
- 年干支：`(年份-4) % 10` 和 `(年份-4) % 12`，已有
- 月干支：根据年份和月份推月建（"甲己之年丙作首"等五虎遁口诀）
- 时干支：根据日干推时干（"甲己还加甲"等五鼠遁口诀）
- 纳音五行：60甲子纳音表，硬编码
- 四柱全展示是亮点，比一般转换器更有深度

### 2.3 简易起卦工具 (divination.html)

**核心功能**
通过"金钱卦"（三枚硬币法）模拟起卦过程，生成卦象并关联到站内六十四卦详解。

**用户场景**
用户想体验传统起卦过程，了解当前卦象的含义，并跳转到详细的卦象解读页面继续学习。

**交互逻辑**

| 操作 | 交互 | 反馈 |
|------|------|------|
| 点击"摇卦" | 按钮点击 | 动画模拟3枚硬币抛掷，显示正面/反面计数 |
| 逐爻生成 | 每抛一次生成一爻 | 从下到上（初爻→上爻）逐步显示6爻 |
| 自动成卦 | 6次抛完后自动计算 | 显示卦名、卦符、卦序编号 |
| 查看详解 | 点击卦名 | 跳转到 hexagrams.html 对应卦的位置（用锚点） |

**页面布局**

```
┌─────────────────────────────────────┐
│  简易起卦                            │
│                                     │
│  ┌─────────────────────────────┐   │
│  │  🪙  三枚硬币抛掷模拟区        │   │
│  │      [点击摇卦]               │   │
│  │                             │   │
│  │  第 1 次:  正正反 → 少阴 ⚋   │   │
│  │  第 2 次:  正反正 → 少阳 ⚊   │   │
│  │  第 3 次:  反反正 → 少阴 ⚋   │   │
│  │  ...                         │   │
│  └─────────────────────────────┘   │
│                                     │
│  成卦结果                            │
│  ┌─────────────────────────────┐   │
│  │  ▄▄▄▄▄ 上九                  │   │
│  │  ▄▄▄▄▄ 九五                  │   │
│  │  ▄▄  ▄▄ 六四                 │   │
│  │  ▄▄▄▄▄ 九三                  │   │
│  │  ▄▄▄▄▄ 九二                  │   │
│  │  ▄▄▄▄▄ 初九                  │   │
│  │                             │   │
│  │  ䷀ 乾为天 · 第1卦            │   │
│  │  → 查看卦象详解               │   │
│  └─────────────────────────────┘   │
│                                     │
│  重新摇卦 [再摇一次]                 │
├─────────────────────────────────────┤
│  本工具仅用于传统文化学术研究场景      │
└─────────────────────────────────────┘
```

**前端实现要点**
- 三枚硬币模拟：`Math.random()*2` 取整，0=正(字)、1=反(花)，三枚求和确定老少阴阳
  - 3正 = 老阳 ○ (变爻)
  - 2正1反 = 少阴 ⚋
  - 1正2反 = 少阳 ⚊
  - 3反 = 老阴 × (变爻)
- 64卦数据：卦名、卦符(unicode)、卦辞、序卦编号，硬编码
- 锚点跳转：hexagrams.html 中的卦卡没有 id，需要加 id 支持（如 `id="hex-1"` ... `id="hex-64"`）
- 变爻提示：如生成老阳/老阴，提示"此爻为变爻"
- 支持"再摇一次"清空重来

### 2.4 五行格局速查 (wuxing-chart.html)

**核心功能**
输入天干和地支，即时查询五行属性、阴阳、纳音、生克关系。

**用户场景**

**交互逻辑**

| 操作 | 交互 | 反馈 |
|------|------|------|
| 选择天干 | 下拉选择 | 五行属性高亮显示 |
| 选择地支 | 下拉选择 | 五行+生肖显示 |
| 组合查询 | 同时选完自动显示 | 完整结果显示：天干+地支、阴阳、五行、纳音 |
| 生克对比 | 输入第二个干支 | 显示两个干支之间的相生相克关系 |

### 2.5 节气与五运六气对照表 (solar-terms.html)

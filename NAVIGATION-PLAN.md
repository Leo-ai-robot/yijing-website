# 易经网站导航重构方案

## 核心策略

主导航固定5个栏目，原有9个平铺页签全部归入二级子菜单。
逻辑沿零基础用户学习路径：首页 → 入门 → 六十四卦 → 五运六气 → 工具

---

## 导航层级清单

### 首页
> 入口门户，展示品牌、干支日历、内容聚合。无二级菜单。

### 易学入门
| 子菜单 | 对应文件 | 说明 |
|--------|----------|------|
| 入门基础 | basics.html | 零基础第一站：什么是易经、成书背景、学习方法 |
| 阴阳八卦 | yinyang-bagua.html | 阴阳哲学、八卦体系、先天后天八卦图 |
| 五行干支 | wuxing-ganzhi.html | 五行生克、天干地支、六十甲子 |
| 六爻入门 | liuyao.html | 卦的构成、爻位含义、承乘比应 |

### 六十四卦
| 子菜单 | 对应文件 | 说明 |
|--------|----------|------|

### 五运六气

## 文案变更对照

| 原菜单文案 | 新菜单文案 | 变更原因 |
|-----------|-----------|---------|
| 入门基础 | 易学入门（主导航）→ 入门基础（子） | 上级栏目标题更抽象，子菜单保留具体 |
| 阴阳八卦 | → 阴阳八卦（子） | 不变，归属到易学入门下 |
| 五行干支 | → 五行干支（子） | 不变，归属到易学入门下 |
| 六爻基础 | → 六爻入门（子） | 改成"入门"强调学习路径连贯 |
| 2026五运六气 | 五运六气（主导航）→ 丙午年专题（子） | 主导航抽象化，子菜单保留年份信息 |
| 每日易学 | 易学工具 → 每日语录 | 归入工具类目，突出每日使用属性 |
| 术语词典 | 易学工具 → 术语词典 | 归入工具类目，速查工具定位 |

## 前端实现要点

- **PC端**：hover展开下拉菜单
- **移动端**：汉堡菜单点击展开 → 有子菜单的项显示▶展开箭头 → 点击展开二级列表
- **高亮**：当前页面所在的主导航项自动加亮（active状态）
- **UI**：保持现有配色和字体体系，只改导航结构

## 四、全站UI视觉规范

### 4.1 配色体系

```css
:root {
    /* ── 基底色 ── */
    --paper: #f5f0e8;           /* 宣纸米白 - 全站底色 */
    --paper-dark: #ebe4d8;      /* 陈纸浅褐 - 卡片/深色区 */
    --paper-light: #faf6ef;     /* 新纸白 - 亮色卡片 */

    /* ── 文字色 ── */
    --ink-black: #1a1a1a;       /* 墨黑 - 主文字/标题 */
    --ink-dark: #2c2c2c;       /* 深墨 - 次标题 */
    --ink-gray: #555555;        /* 淡墨 - 正文 */
    --ink-light: #888888;       /* 灰墨 - 辅助文字 */

    /* ── 强调色 ── */
    --cinnabar: #8b4513;        /* 暗赭石 - 链接/按钮/强调（似印泥色） */
    --cinnabar-dark: #6b3410;   /* 深赭 - hover */
    --cinnabar-light: #a05a2c;  /* 浅赭 - 轻高亮 */

    /* ── 辅助色 ── */
    --slate-gray: #8a9ba8;      /* 浅青灰 - 副标题/辅助信息 */
    --slate-light: #d5dde3;     /* 淡青灰 - 分割线/边框 */
    --dark-gold: #b8860b;       /* 哑金 - 公告栏点缀（极少用） */
    --dark-gold-light: #d4a017;

    /* ── 边框 ── */
    --border-light: #ddd6c8;    /* 淡茶色 - 通用边框 */
}
```

### 4.2 字体规范

| 层级 | 字体栈 | 字重 | 行高 | 字间距 | 字号 |
|------|--------|------|------|--------|------|
| 大标题 H1 | `ZCOOL XiaoWei` / 楷体 | 400 | 1.3 | 0.05em | 2-2.8rem |
| 中标题 H2 | `ZCOOL XiaoWei` / 楷体 | 400 | 1.4 | 0.04em | 1.5-2.2rem |
| 小标题 H3/H4 | `Noto Serif SC` / 宋体 | 700 | 1.5 | 0.02em | 1.1-1.5rem |
| 正文 | `Noto Serif SC` / 宋体 | 400 | 1.9 | 0.01em | 0.9-1rem |
| 辅助文字 | `Noto Sans SC` / 黑体 | 300/400 | 1.6 | 0.02em | 0.75-0.85rem |
| 导航/按钮 | `Noto Sans SC` / 黑体 | 400 | 1.4 | 0.06em | 0.82-0.9rem |

### 4.3 间距体系

```css
--space-xs: 0.5rem
--space-sm: 1rem
--space-md: 2.5rem
--space-lg: 3.5rem
--space-xl: 6rem
--max-width: 1200px
--content-width: 800px
```

留白原则：内容区窄化居中（800px），左右留白充裕。段落间距1.4rem，H2底部增加60px赭石装饰线。

### 4.4 太极元素使用规则

| 位置 | 元素 | 规格 | 透明度 |
|------|------|------|--------|
| 分割线 | 太极图标居中+渐变线 | 32×32 SVG | 25% |
| Logo | 太极双鱼徽标 | 40×40 SVG | 60% |
| Hero装饰 | 同心圆环 | 60×60 | 60% |
| 卡片hover | 太极暗纹（右下角） | 40×40 | 8% |
| Tool浮窗 | ☯ Unicode字符 | 1.2rem | 100% |

### 4.5 卡片统一样式

```css
.card {
    background: var(--paper);
    border: 1px solid var(--border-light);
    border-radius: 8px;
    padding: var(--space-md);
    transition: all 0.3s ease;
}
.card:hover {
    border-color: var(--cinnabar);
    box-shadow: 0 2px 12px rgba(139, 69, 19, 0.06);
}
```

### 4.6 背景纹理

- 宣纸底色 #f5f0e8
- 噪点纹理（fractalNoise 0.9, opacity 0.03）
- 左上暖色径向渐变（rgba(139,69,19,0.03)）
- 右下冷色径向渐变（rgba(138,155,168,0.03)）
